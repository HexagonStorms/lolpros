FREQUENTLY ASKED QUESTIONS

What is this site?

Inspired by Novocalculus, this site is a directory of Twitch streams for each Professional League of Legends player across all major regions. This site also provides details of the players, including image, bio, and other miscellaneous information.

What technologies were used?

Frontend: AngularJS, HTML5, SASS, Bootstrap 3, Twitch API (v2)
Backend: Laravel 5.5, PHP 7.1, MySQL, (LAMP Stack)

How often is this site updated?

I follow the eSports scene obsessively, so I try to update this constantly. I built a few cronjobs that can update automatically by sweeping the Official eSports site as well as decoding the unofficial Riot eSports API. However, it is prone to error and I often update information manually. Please contact me if you see anything out of date!
